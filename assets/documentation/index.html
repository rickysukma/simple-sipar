HTML Structure

The template is based on Foundation Framework - http://foundation.zurb.com/

Foundation is now Mobile First, meaning you start building for small devices first, then layer in more complexity as devices become larger and more powerful.

The grid allows for nesting down as far as you'd like, though at a certain point it will get absurd. You can use this nesting to create quite complex layouts, as well as some other tricks like form layouts or visual elements.

Javascript

For more information, please visit http://www.jquery.com/

The initialization of the elements, libs and features is made by the file script.js, in the JS folder.
